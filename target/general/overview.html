CodeLatch - Target Overview

A target is the destination for your code. Generally this will be a particular microcontroller and circuit board combination. Target profiles detail the various tools, libraries, and options needed to build code for a given target. The current profile is selected from the main window toolbar.



Use the target profile selection combo box to select the current profile. Pressing the target edit button will activate the profile editor. While the editor is active you can change the profile selection combo box to view other profiles. Profiles are automatically saved any time you make a change.

For example the OLIMEXINO-STM32. This is a board designed by Olimex and can be purchased from a variety of store fronts.
