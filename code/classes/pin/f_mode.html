Pin.mode(pin,mode) - Sets the operating mode of the specified pin.

Parameters

pin - The name of the pin to configure. Pin names are defined in a library header file for each target. Please see the documented targets for specific pin names. In general there are definitions for physical IC pins and target defined digital and analog pins. They are specified by a P, D, or A followed by a number.

P1..Pxx - microcontroller physical pins.
D0..Dxx - digital pins defined for a given target.
A0..Axx - analog pins defined for a given target.

mode - The mode of the pin, mode can be one or a valid combination of the following:

OUTPUT
INPUT
PULLUP
PULLDOWN
OPENDRAIN
NORMAL
HYSTERESIS
ADC
ALT
PWM

Return: none.

Example Usage

Pin.mode ( D3, INPUT ); // make pin D3 a digital input.
Pin.mode ( D3, INPUT | PULLUP ) ; // make pin D3 a digital input with a pull-up.
Pin.mode ( D3, OUTPUT ); // make pin D3 a digital output.
Pin.mode ( D3, OUTPUT | OPENDRAIN ); // make pin D3 a digital output with an open-drain drive.
Pin.mode ( D1, PWM ); // make pin D1 a PWM output.
Pin.mode ( A2, ADC ); // make pin A2 an ADC input.
Pin.mode ( P8, OUTPUT); // make pin 8 of the microcontroller a digital output.
